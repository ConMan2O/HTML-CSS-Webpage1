Welcome to the World of HTML & CSS!

Introduction

What is HTML?

HTML is the language used to create and structure content on web pages.

What is CSS?

CSS is the language that styles and designs how web pages look.

HTML Basics

What is a tag?

A tag is a code element in HTML that defines parts of a webpage, like headings, paragraphs, or images.

What is an element?

An element is a complete HTML structure made up of a start tag, content, and an end tag.

What is an attribute?

An attribute is extra information inside an HTML tag that defines properties or behavior.

CSS Basics

What is a selector?

A selector is a pattern in CSS that targets HTML elements to apply styles to them.

What is a class?

A class is a label you give to HTML elements so you can style or control groups of them with CSS or JavaScript.

What is an ID?

An ID is a unique identifier given to a single HTML element for styling or scripting purposes.
